## 📖 Overview

This portfolio was designed with one objective:

> **Create a memorable developer portfolio that feels more like a modern product experience than a traditional personal website.**

Instead of relying on generic templates, the project combines cinematic visuals, interactive animations, and responsive design to provide an engaging user experience while highlighting projects, skills, certifications, and technical expertise.

# 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/tsaksham1304/portfolio.git
```

### Navigate into the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
